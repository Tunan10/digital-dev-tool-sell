# 🚀 DegiTools

---

## 📌 Description

DegiTools is a modern and responsive web application where users can explore
digital products, subscribe to plans, and manage their cart in real-time.

---

## 🛠️ Technologies Used

- ⚛️ React.js
- 🎨 Tailwind CSS
- 🌼 DaisyUI
- 📦 JavaScript (ES6+)

---

## ✨ Features

- 🛒 Real-time cart update with navbar badge
- 🎯 Responsive and interactive pricing section
- ⚡ Clean UI with smooth animations
